# ⭐ Sistema de Pesquisa de Satisfação (NPS)

Sistema web desenvolvido para coletar e analisar a satisfação dos clientes através da metodologia NPS (Net Promoter Score).

A aplicação permite que o cliente avalie o serviço prestado, envie comentários e registre sua nota de forma rápida e intuitiva. As respostas são armazenadas automaticamente no Google Sheets e utilizadas para geração de indicadores e dashboards.

## 📊 Sobre o NPS

O Net Promoter Score (NPS) é uma metodologia utilizada para medir a lealdade e satisfação dos clientes.

As respostas são classificadas em:

- **Promotores** → notas 9 e 10  
- **Neutros** → notas 7 e 8  
- **Detratores** → notas de 0 a 6  

Fórmula:

NPS = % Promotores - % Detratores
